# Bioinformatics Sandbox & Project Archive

Welcome to my technical playground! This repository serves as a centralized archive for independent learning modules, algorithm challenges, and collaborative script contributions.

---

## 📁 Repository Structure

### 🧬 [DNA Sequence Parser](./dna_sequence_parser)
* **Description:** A Python utility for processing raw FASTA files.
* **Core Functions:** Automates GC-content calculations, filters low-quality sequences, and generates quality control metrics.

### 🤖 [HPO Data Mapper](./hpo_data_mapper)
* **Description:** An NLP-based text-mining tool designed to parse unstructured clinical notes.
* **Core Functions:** Extracts and maps phenotypic descriptions to standardized **Human Phenotype Ontology (HPO)** terms to aid in downstream diagnostic pipelines.

### 🤝 [Vibiome Summit Hackathon Contribution](./vibiome_hackathon)
* **Description:** Collaborative code adjustments submitted via Pull Request during the Vibiome Summit event.
* **Core Functions:** Minor code optimizations to streamline real-time sequence tracking and ensure data compatibility between team modules.

---

## 💻 Tech Stack
* **Languages:** Python (Pandas, Biopython)
* **Environment:** Linux / WSL
* **Data Standards:** FASTA, HPO
